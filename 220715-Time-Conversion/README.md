# Time Conversion Algorithm Read Me

# Description

Given a time in 12-hour AM/PM format, convert it to military (24-hour) time.

# Language

JavaScript (Node.js)

# Origin

HackerRank

# Example

```
s='12:01:00PM'

return '12:01:00'
```

```
s='12:01:00AM'

return '00:01:00'
```

```
s='07:05:45PM'

return '19:05:45'
```
