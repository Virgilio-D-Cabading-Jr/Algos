# Comtains Duplicate Algorithm Read Me

## Description

Give an integer array (nums), return true if any value appears at least twice in the array, and return false if every element is distinct

## Language

JavaScript

## Origin

LeetCode

## Examples

Input: nums = [1, 2, 3, 1]
Output: True

Input: nums = [1, 2, 3, 4]
Output: False

Input: nums = [1, 1, 1, 3, 3, 4, 3, 2, 4, 2]
Output: True
